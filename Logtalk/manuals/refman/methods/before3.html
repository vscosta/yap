before/3

Description

before(Object, Message, Sender)

This is a pre-declared but user-defined public method for handling before events.

Template and modes

before(?object, ?term, ?object)

Errors

(none)

Examples

before(Object, Message, Sender) :-
    writeq(Object), write('::'), writeq(Message),
    write(' from '), writeq(Sender), nl.
