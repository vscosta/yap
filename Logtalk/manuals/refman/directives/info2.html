info/2

Description

info(Functor/Arity, List)

Documentation directive for predicates.

Template and modes

info(+predicate_indicator, +info_list)

Examples

:- info(empty/1, [
    comment is 'True if the argument is an empty list.',
    argnames is ['List']]).
