Welcome to the Logtalk for Mac OS X Installer

Logtalk is an open source object-oriented extension to the Prolog programming language. Integrating logic programming with object-oriented and event-driven programming, it is compatible with most Prolog compilers. It supports both prototypes and classes. In addition, it supports component-based programming through category-based composition.

This installer guides you through the steps necessary to install Logtalk for Mac OS X. Logtalk is installed on the /opt/local/share/logtalk directory. After completing the installation, each Logtalk end-user must run the shell command cplgtdirs once before using the Logtalk – Prolog integration scripts. This command creates a folder named logtalk on the user's home directory containing configuration files, examples, and documentation. Inside this folder is a link to a CUSTOMIZE.txt file describes how to customize Logtalk for your personal use. Links to the Logtalk – Prolog integration scripts are created on the /opt/local/bin directory, which must be on your system path. The integration scripts are described on the /opt/local/share/logtalk/scripts/NOTES.txt file.
